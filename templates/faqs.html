• Q: How can I add a class to get notified when it is open? A: Navigate to app [will set up the home page link here] and just fill out the form there and you will be all set.
  • Q: How do I stop the receiving text messages for requested class(es)? A: You can do one of the followings to stop getting text messages. Regardless of the choice you make, we will send you a text saying that we have deleted your requested class. If you do not get a text message saying we have deleted it then that means you did not have a such request. 1. Completely Stop Getting Text Messages: you can simply reply to the message saying: cancel all. You MUST NOT write anything else before or after cancel all. It does not matter if the text cancel all is upper case or not. i.e you can still add classes and get notified later on even if you take this approach. 2. Stop Notification for Only One Class: you can simply reply to the message saying: cancel [5-digit-class-code]. ex. cancel 45678
    You can find the classes you requested in the chat. You MUST NOT write anything else before or after cancel [5-digit-class-code]. It does not matter if the text cancel [5-digit-class-code] is upper case or not.
    This is a convenient way to stop getting notification as it allows you to stop getting notification for a class which you may have found a seat or not taking this class anymore or decided to get notification for the same course but a different section. i.e you can still add classes and get notified later on even if you take this approach.
  • Q: How does this application work in regards to search if a class is closed or not? A: This application has a scheduler which runs in every 2 minutes. The scheduler gets all the request(s) (e.g. request mean the classes students have entered to get notified when it opens) available and creates threads and search each class. If any of the class is found open then those who have made the request for this class will get notified right away in couples of seconds. Keep it in mind that if CUNYfirst is down due to maintenance then we will not be able to search your requested class(es) until it is back online.
