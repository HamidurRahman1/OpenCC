Frequently Asked Questions

 Navigate to the OpenCC application and just fill out the form there with the correct information and you will be all set.
 You can do one of the followings to stop getting text messages. Regardless of the choice you make, a text will be sent saying that your requested class(es) have been deleted. If you do not get a text message that means you did not have a such request to be deleted.

  1. Completely Stop Getting Text Messages:

     You can simply reply to the message saying: cancel all. You MUST NOT write anything else before or after cancel all. It does not matter if the text cancel all is upper case or not as long as the replied message only contains cancel all (i.e you can still add classes and get notified even if you take this approach.)

  2. Stop Getting Text Messages for Only One Class:

     You can simply reply to the message saying: cancel [5-digit-class-number]. ex. cancel 45678. You can find the classes you requested in the chat. You MUST NOT write anything else before or after cancel [5-digit-class-code]. It does not matter if the text cancel [5-digit-class-code] is upper case or not.
     This is a convenient way to stop getting notification as it allows you to stop getting notification for a class which you may have found a seat or not taking this class anymore or decided to get notification for the same course but a different section (i.e you can still add classes and get notified later on even if you take this approach.)
 This application has a scheduler which runs in every 2.5 minutes. The scheduler gets all the available requests (e.g. request mean the classes students have entered to get notified when it opens) and creates threads and search each class. If any of the class is found open then those who have made the request for this class will get notified right away in couples of seconds. Keep it in mind that if CUNYfirst is down due to maintenance then we will not be able to search requested class(es) until it is back online. In case if a class become open exactly right before CUNYfirt goes under maintenance, no one will be able to get that class until CUNYfirst is back online. As the scheduler runs in every 2.5 minutes, you still have a very high chance of getting this class if CUNYfirst is back online after maintenance.
